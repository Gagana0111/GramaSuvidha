# GramaSuvidha 🏘

> ಗ್ರಾಮ ಡಿಜಿಟಲ್ ಪೋರ್ಟಲ್ | Village Digital Portal

A bilingual Android application for Karnataka Gram Panchayat digital project tracking.

## 📱 Features

- ✅ Real-time village project tracking
- ✅ Bilingual support — Kannada & English
- ✅ Complaint filing with auto-generated IDs
- ✅ Citizen polls & voting
- ✅ Budget tracker with charts
- ✅ Photo gallery for project progress
- ✅ Firebase backend (Firestore, Auth, Storage)
- ✅ Gemini AI integration
- ✅ Google Maps for project locations
- ✅ 16 fully functional screens

## 🛠 Tech Stack

| Technology | Purpose |
|---|---|
| Kotlin | Primary Language |
| Jetpack Compose | UI Framework |
| Firebase Firestore | Real-time Database |
| Firebase Auth | Authentication |
| Firebase Storage | Photo Storage |

Photo Gallery → Budget Tracker → Announcements → Polls →
Notifications → Complaint → Profile → Settings → Search →
Firebase Demo → Maps

## 🚀 Setup

1. Clone the repository
2. Open in Android Studio
3. Add your `google-services.json` from Firebase Console
4. Add your Gemini API key in `local.properties`
5. Run on emulator or device (API 24+)

## 📋 Requirements

- Android Studio Meerkat or later
- Minimum SDK: API 24 (Android 7.0)
- Target SDK: API 35
- Kotlin 2.0.0
